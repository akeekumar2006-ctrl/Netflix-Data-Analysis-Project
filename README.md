# Netflix Data Analysis Project

This project analyzes Netflix titles data using Python, Pandas, and Matplotlib.

## Project Overview

In this project, I performed data analysis and visualization on Netflix content data to understand different trends and patterns.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Visualizations Created

- Movies vs TV Shows
- Rating Distribution
- Movie Duration Distribution
- Movies vs TV Shows by Year
- Releases Over the Years
- Top 10 Countries with Netflix Content

## Dataset

The dataset used in this project is `netflix_titles.csv`.

## Key Learnings

Through this project, I practiced:

- Loading and reading CSV data
- Data cleaning
- Data analysis using Pandas
- Creating charts using Matplotlib
- Presenting insights through visualizations

## Conclusion

This project helped me improve my data analysis and data visualization skills using Python.
